# Sheba Clinic Pharmacy Management System

A comprehensive web-based Pharmacy Management System developed using PHP, MySQL, JavaScript, HTML, and CSS. The application is designed to streamline pharmacy operations by providing efficient medicine inventory management, order processing, stock monitoring, and customer service functionalities through dedicated administrative and customer interfaces.

## Key Features

### Administrative Module

* Secure Admin Authentication
* Interactive Dashboard
* Medicine Management (Create, Update, Delete)
* Category Management
* Inventory Monitoring
* Stock Request Management
* Customer Order Processing
* Reporting and Analytics

### Customer Module

* User Registration and Authentication
* Medicine Browsing and Search
* Shopping Cart Functionality
* Order Placement and Checkout
* Real-Time Order Tracking

## Installation Guide

### Prerequisites

* XAMPP / WAMP Server
* PHP 7.4+
* MySQL

### Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/AhmmedRizvi/Pharmacy_Management_System.git
```

2. Move the project folder to the web server directory (`htdocs` for XAMPP).

3. Create a MySQL database.

4. Import the provided database file:

```text
shebaclinic.sql
```

5. Configure database credentials in:

```text
config.php
```

6. Start Apache and MySQL services.

7. Open the application in your browser:

```text
http://localhost/Pharma_Proj
```

## Project Objectives

The primary objective of this project is to digitize pharmacy operations by providing a centralized platform for medicine inventory management, customer order processing, and stock control. The system enhances operational efficiency, reduces manual workload, and improves the overall customer experience.
